## About

Interactive personal portfolio built as a single-file HTML experience — featuring an animated boot sequence, live NOC dashboard, interactive skill network graph, in-browser terminal, and 3D project cards. Designed to showcase both network engineering and software development capabilities.

## Tech stack

- **Pure HTML, CSS, JavaScript** — no build step, no dependencies
- **Canvas API** — animated particle network background, live throughput chart
- **SVG** — interactive skill graph, animated network globe
- **CSS animations** — boot sequence, scroll reveals, glitch & gradient effects
- **Google Fonts** — Space Grotesk, Inter, JetBrains Mono

## Features

- 🎬 Animated boot sequence on load
- 🖱️ Mouse-reactive cursor glow and particle network
- 🧠 Interactive skill graph (hover for details)
- 📊 Live NOC dashboard with real-time metrics, throughput chart, and event log
- 🌐 Animated network globe with connection arcs
- 💳 3D-tilt project cards
- 💻 Built-in terminal with commands (`help`, `about`, `ping`, `scan`, `sudo clearance` 🤫)
- 📱 Fully responsive · ⚡ Zero dependencies · 🌙 Dark mode native

## Local development

Just open `index.html` in any modern browser. No build step required.

```bash
# Or serve with any static server
python3 -m http.server 8000
# Then visit http://localhost:8000
```

## Deployment

Deployed via GitHub Pages on every push to `main`.

GitHub Pages should be configured to use the `GitHub Actions` source so the workflow in `.github/workflows/deploy-pages.yml` can publish the site.
